# Financial Data Cleaning and Analysis

Author: Nicholas Jojola

This project demonstrates a financial data analytics pipeline implemented in R.

Key features:

- Data cleaning and duplicate detection
- Grouped statistical summaries by state
- Identification of anomalous financial ratios
- Feature engineering (debt-to-income metric)
- Visualization of financial trends
- Algorithm documentation via flowchart and pseudocode

The complete analysis report is available in the HTML document.
